# Roadmap

This roadmap keeps the starter pack focused on making file-based AI work understandable, safe, and repeatable.

## v0.1.0 - First Public Starter Pack

Status: prepared.

Goals:

- public README explains the method in 30-120 seconds;
- quickstart gives a safe first test;
- license and contribution files are present;
- one minimal example shows the first-session workflow;
- local validation scripts run without external services.

## v0.2.0 - Feedback And First External Examples

Planned:

- collect first-run feedback from early users;
- add at least one more realistic example;
- improve issue templates and discussion prompts;
- add a checklist for adapting the starter pack to an existing Markdown vault;
- document known limitations.

## v0.3.0 - Tool Compatibility Notes

Planned:

- add tested setup notes for specific file-aware agents;
- describe what each tool can and cannot safely edit;
- add a compatibility report template;
- only then add tool-specific GitHub topics.

## Later

Possible:

- stronger validation scripts;
- more example project types;
- migration guide for existing Obsidian vaults;
- smaller distribution profile for people who only want the core method;
- community examples, if users contribute them.

## Non-Goals

- becoming a hosted memory product;
- replacing Obsidian, GitHub Issues, or project trackers;
- promising universal AI memory;
- adding automation before the manual method is clear.
