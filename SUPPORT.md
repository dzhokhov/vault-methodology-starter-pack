# Support

This is an early open-source starter pack. Support happens through GitHub issues and discussions.

## Ask A Question

Open a GitHub Discussion for:

- first-run questions;
- adaptation questions;
- "where should this file live?" questions;
- tool compatibility notes;
- examples of your own folder structure.

## Report A Problem

Open a GitHub Issue when:

- a Markdown link is broken;
- a script fails;
- a public file mentions private or non-portable material;
- quickstart instructions are unclear;
- a template leads to unsafe or confusing agent behavior.

Please include:

- what you tried;
- which file or instruction you followed;
- what happened;
- what you expected.

## Security And Privacy

Do not post private vault contents, tokens, customer data, personal records, or private company material in public issues.

If you find private material that appears to have been accidentally included in the starter pack, open an issue with the filename and a short description. Do not quote the private text unless it is necessary to locate the problem.

## Scope

This repository does not provide:

- hosted support;
- guaranteed compatibility with every AI assistant;
- automatic migration from an existing vault;
- professional services.

The best support request is a small, reproducible first-run problem.
