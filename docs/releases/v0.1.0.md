# Release v0.1.0

First public starter-pack release.

## What This Release Is

`v0.1.0` is an early public baseline for testing an agent-ready Markdown vault:

- public README;
- quickstart;
- license;
- contribution and support files;
- roadmap and changelog;
- issue templates;
- first minimal example;
- local validation scripts.

## Who Should Try It

Try this release if you already use file-aware AI agents, Markdown notes, or Obsidian, and you want project state to survive across sessions.

## How To Try It

1. Use the repository as a template or clone it into a safe test folder.
2. Read `README.md` and `QUICKSTART.md`.
3. Put a small test note in `00_inbox/`.
4. Ask your agent to route it into a new project and update links.
5. Compare the result with `examples/first-session/`.

## Known Limits

- No broad compatibility claims yet.
- No external user examples yet.
- No hosted application or plugin.
- The safest first run is in a copy, not your real vault.
