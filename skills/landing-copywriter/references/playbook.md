## 7-шаговый протокол Hero (краткая версия)

1. **Стадия Schwartz.** Определи стадию осведомлённости трафика:
   unaware / problem / solution / product / most. Один Hero ≠ на все стадии. По умолчанию
   для холодного B2B SaaS — solution-aware.
2. **Ядро позиционирования (Trout/Ries).** Одно предложение:
   «[Продукт] — это [категория] для [кто], которая [выгода], в отличие от [альтернатива],
   потому что [дифференциатор]». Лакмус: подставь конкурента — звучит? Если да, переписывай.
3. **10 кандидатов H1.** Разный диапазон: PAS/AIDA, активный глагол, триада, контраст, число,
   brand-ownable провокация. Цель — диапазон, а не «один шедевр».
4. **Прогон через 5 фильтров.** Visualize / Falsify / Brand-ownable / Ритм+clarity / RU-фильтр.
   Из 10 остаётся 3 финалиста.
5. **Subhead.** 1–2 строки, 16–25 слов в русском. Не повторяет H1 — несёт новый квант:
   доказательство, выгода, снятие возражения. Конкретика обязательна.
6. **Hero как блок.** H1 + subhead + proof + primary CTA + visual hint. Один primary CTA.
   CTA — глагол + объект, 2–4 слова. Proof — конкретный (цифра/имя/KPI).
7. **Чеклист валидации.** 5-second test, three-question test (Harry Dry), лакмус Trout/Ries,
   message-match, RU-фильтр.

## Schwartz: 5 стадий и что говорить

| Стадия | Что знает | Hero говорит о |
|--------|-----------|----------------|
| Unaware | Не знает о проблеме | Контексте, hook, провокации (НЕ продукт) |
| Problem-aware | Знает проблему, не решения | Проблеме + обещание решения |
| Solution-aware | Знает решения, не твой продукт | Выгоде + почему твой тип решения лучше |
| Product-aware | Знает продукт, не уверен | Дифференциаторе, proof, снятии возражений |
| Most-aware | Хочет купить | Оффер, срочность, CTA |

## Ogilvy headline rules (применимые)

- Указывай аудиторию в заголовке (роль/индустрия/размер).
- Никаких каламбуров, аллюзий, игры слов в B2B без awareness-бюджета.
- Цифры > общих слов.
- Заголовок должен быть самодостаточен без body copy.
- Активный глагол лучше существительного.

---

## Harry Dry — three-question test

1. **Visualize:** видна картинка в голове за 1 секунду?
2. **Falsify:** проверяемо как true/false?
3. **Unique:** конкурент не может сказать то же?

Три «да/да/нет» — заголовок сильный.

## RU-фильтр чистоты текста

**Стоп-слова и кальки, которые провалят любой Hero:**
- решения (как синоним продуктов)
- оптимизация
- бесшовный, бесшовная интеграция
- единая экосистема
- инновационный, инновационная платформа
- уникальный, эффективный, качественный
- комплексный подход
- передовой, ведущий

**Правила:**
- Глаголы > причастий («Мы платим» > «Осуществление выплат»).
- Конкретика > оценочности («3 минуты» > «быстро»).
- Одна мысль = одно предложение.
- В русском Hero хорошо работает 5–9 слов.
- Активный глагол в начале — мощный приём.

## Формулы — шпаргалка

**PAS (Problem → Agitation → Solution).** Для unaware/problem-aware. Эмоциональнее AIDA.

**AIDA (Attention → Interest → Desire → Action).** Для горячего трафика в buying mode.

**4U (Urgent, Unique, Useful, Ultra-specific).** Чеклист для отдельной фразы, не структура.

## Технические приёмы «звонкости» (без потери clarity)

- **Триада:** «Реклама. Гонорары. Отчёты.»
- **Инверсия:** «Растёт вместе с вами — а не за ваш счёт.»
- **Пауза через тире/точку:** «Один click. Десять договоров забудьте.»
- **Конкретная метонимия:** «Один акт» вместо «упрощение документооборота».
- **Активный глагол в начале:** «Платите одним актом».
- **Число вместо прилагательного:** «В 4 раза быстрее».

## Принципы, которые не нарушаются

- **Clarity > cleverness** в B2B SaaS без awareness-бюджета (~85–90% A/B-кейсов).
- **Brand-ownable — обязательный фильтр.** Лакмус Trout/Ries проходит каждый кандидат.
- **Один Hero = один квант смысла.** Несколько value props = несколько лендингов.
- **Стадия Schwartz — это решение, а не теория.** Без неё пишешь вслепую.
- **Русский ≠ английский.** Кальки делают текст слабее и менее конкретным.

## Минимум входа от пользователя

Без этих 4 пунктов Hero писать нельзя — задавай уточняющий вопрос:

1. Что за продукт (1–2 предложения).
2. ICP / роль / индустрия.
3. Источник трафика (или хотя бы стадия Schwartz).
4. 1–2 дифференциатора от альтернатив.
