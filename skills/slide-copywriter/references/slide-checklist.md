# Чеклист качества контента слайдов

Пройди каждый пункт перед тем, как отдать результат пользователю.

## Уровень деки (макроструктура)

- [ ] SCQA сформулирована и согласована с пользователем
- [ ] Ответ (Answer) на слайде 1-2, не в конце
- [ ] Тест «только заголовки»: прочитай все заголовки подряд — получается связная история?
- [ ] Есть чередование «что есть ↔ что может быть» (Duarte Sparkline)
- [ ] Каждый слайд ведёт к выводу или решению (нет «информационных» слайдов без action)
- [ ] Общий объём: 15-25 слайдов для live-презентации
- [ ] Если данных больше, чем влезает — есть handout / appendix

## Уровень слайда (микроструктура)

- [ ] Заголовок = вывод (action title), а не описание темы
- [ ] Заголовок ≤12 слов
- [ ] Тело: ≤6 визуальных элементов (Rule of Six)
- [ ] Одна мысль на слайд (исключения: comparison, scorecard)
- [ ] Нет полных абзацев на слайде — только ключевые фразы, цифры, графики
- [ ] Указан тип визуализации (таблица / график / схема / bullet-поинты)
- [ ] Speaker notes: 3-5 предложений (контекст + переход + предвосхищение вопросов)

## Чистота текста (Ильяхов для слайдов)

- [ ] Нет вводных конструкций («следует отметить», «важно подчеркнуть»)
- [ ] Нет навязанных оценок без фактов («уникальный», «эффективный», «инновационный»)
- [ ] Нет канцелярита («осуществлять», «в целях», «в рамках», «на основании»)
- [ ] Нет неопределённости («некоторый», «ряд», «различный») — заменено на цифры
- [ ] Действительный залог вместо страдательного
- [ ] Bullet-поинты по формуле: [Что] + [Насколько] + [Так что]
- [ ] После удаления мусора текст наполнен фактами (не стал сухим)

## Data-слайды

- [ ] Таблица ≤6 строк (остальное в handout)
- [ ] Есть RAG-индикаторы / цветовая кодировка для быстрой считываемости
- [ ] Убраны сетки, 3D-эффекты, декоративные элементы
- [ ] Вывод из данных — в заголовке, данные — как proof в теле

## Финальная проверка

- [ ] Нет антипаттернов: описательных заголовков, детективной интриги, copy-paste таблиц, слайдов «обо всём»
- [ ] Каждый слайд отвечает на вопрос «So what?»
- [ ] Если это стратсессия — каждый слайд ведёт к развилке или рекомендации
