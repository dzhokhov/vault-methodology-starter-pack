# Editorial Checklist

> Навигация: [SKILL](../SKILL.md)

## Clarity
- Тема ясна из первого абзаца.
- Нет необъяснённых терминов.
- Одна мысль на абзац.

## Practicality
- Есть прикладной эффект на процесс/роль.
- Есть действие в горизонте 7 дней.
- Есть ограничения/риски неверного применения.

## Language
- Нет канцелярита и маркетинговых штампов.
- Нет утверждений "лучший/революционный" без доказуемой механики.
- Нет пассивных формулировок, скрывающих ответственность.
